Design the same website we have, but using React and Django with postgresql as database.


In one corner of the website, place a Sign Up and Login button.

When the user clicks Sign Up, ask for four things (no repetition required for the username):

Username
Phone number
Email address
Password
Confirm password

Store all of this information and then return the user to the main page of the website.

When the user clicks Login, ask only for:

Username
Password

If the information is correct, log the user into the site.

All articles on the website should send their visit counts to our backend.

There should be three access levels:

The first admin is me, and I have access to everything.
The second level is for coaches.
The third level is for users who log into the system as described above.

In another section of the website, create a Coach Registration button.

After clicking it, coaches should fill out an online form containing:

First name
Last name
National ID number
Date of birth
Educational documents
Digital signature
Any other required information

They then submit the form.

After I approve it, they should gain coach-level access.

One important point: before someone can register as a coach, they must first have registered as a normal user.

After logging into the site, the user's first and last name should appear at the top of the page.
