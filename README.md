# روانشناسی ورزشی

این پروژه یک وب‌سایت جامع درباره روانشناسی ورزشی است که شامل مقالات آموزشی، اطلاعات درباره انواع ورزش‌ها، و یک سیستم مدیریت قرار ملاقات است.

## ساختار پروژه

- `backend/`: برنامه Django برای مدیریت قرار ملاقات‌ها و کاربران
- فایل‌های HTML: صفحات وب‌سایت
- `static/`: فایل‌های استاتیک (CSS, JS, تصاویر)

## راه‌اندازی

### Backend

1. نصب وابستگی‌ها:
   ```bash
   cd backend
   pip install -r requirements.txt
   ```

2. اجرای migrationها:
   ```bash
   python manage.py migrate
   ```

3. اجرای سرور:
   ```bash
   python manage.py runserver
   ```

### Frontend

فایل‌های HTML را در یک وب سرور محلی serve کنید یا مستقیماً باز کنید.

## ویژگی‌ها

- مقالات آموزشی درباره روانشناسی ورزشی
- اطلاعات درباره انواع ورزش‌ها
- سیستم مدیریت قرار ملاقات
- طراحی responsive
